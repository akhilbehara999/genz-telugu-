# Change Log

All notable changes to the "genz-telugu" extension will be documented in this file.

## [1.0.0] — 2026-05-20

### Added
- Full syntax highlighting for `.gzt` files
- Auto-completion for all 28 GenZ Telugu keywords
- Hover information showing Python translations
- 17 code snippets (`ifgzt`, `forgzt`, `fungzt`, etc.)
- Run GenZ Telugu file command (`Ctrl+Enter`)
- Translate to Python command (`Ctrl+Shift+T`)
- Keywords reference panel
- Error decorations in editor
- Output panel for execution results
- Dark and light theme compatibility
- Bracket matching and auto-closing
- Comment support (`#`)
- Indentation rules for GenZ Telugu syntax
- Configurable Python path setting
- Auto-translate toggle setting
- Cross-platform support (Windows, macOS, Linux)
